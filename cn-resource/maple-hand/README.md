## Maple Hand

クレ pro 英数微调 + 汉仪唐美人 + 方正大签字笔体

文件为 Magisk 刷机包，使用 Maple Hand 作为基本字体，Maple Mono 作为等宽字体

**MIUI 需刷 空字体模块**

### 获得字体文件

1. 下载 `Maple Hand + Maple Mono.zip`
2. 解压
3. 进入 `system/fonts`
4. 提取 `MapleHand-Regular.ttf` / `MapleHand-Bold.ttf`
